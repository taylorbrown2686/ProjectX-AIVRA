Assign a setting to the "Displacment" Texture: - Wrap Mode - Repeat

Shader setting:

DistortionUV_Wave X,Y - number of waves
DistortionUV_Distance X,Y - wave power
DistortionUV_Speed - speed wave
FishEyeUV_Size -  setting up a circular sphere
ResizeUV X,Y - ability to move your texture
ResizeUV_Zoom X,Y - tiling texture
DisplacmentPack_Value X,Y - displacment speed the cordinats

PositionUV X,Y -  the main control option field

DisplacmentPack_1(RGB) - you texture "tiling". 

TingRGBA_Color_1 - Color you displacment texture
TingRGBA_Color_2 - Color end sphere
TingRGBA_Color_3 - Color sphere

FadeTo Alpha - alpha setting you texture
Mul Fade - increasing the color value for the alpha texture

NewTex_2 - texture UI panel
NewTex_3 - texture where it will effect

NewTex_4 and 1 - creates a gradient
